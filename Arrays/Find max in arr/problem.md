##Find Maximum in an Array

You are given a list of integers representing the heights of mountains. Your task is to find the height of the tallest mountain.

>Input:
The first line contains an integer T, the number of test cases.
For each test case:
The first line contains an integer N, the number of mountains.
The next line contains N space-separated integers, where each integer represents the height of a mountain.

>Output:
For each test case, print a single integer: the height of the tallest mountain.

>Constraints:
1≤T≤10
1≤N≤100000

>input example:
1
5
4 7 6 3 1

>output:
7
